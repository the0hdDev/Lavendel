# [Velt Game Engine](https://Velt.tech)

 Velt is a 2D/3D Game Engine powered by Vulkan and inspired by Game Engines Like [Hazel](hazelengine.com), [Unity](uinty.com) and the [Source](vavtesoftware.com) Engine. The Editor and Debug UI are going to be powered by Dear IMGui or Walnut. 

Currently its just me working on Velt but everyone can Contribute for now. Velt right now is Licensed under BSD-3 Clause but I might switch to another License

## Roadmap - [Velt.tech/Roadmap](Velt.tech/roadmap)
### Velt is currently in very early Development but it definitly have:
ECS based on entt,
2D and 3D Vulkan Renderer Backend,
Physics System based on Box2D,
SDL support
